# Tweets Against Humanity

Extension/Userscript to turn Twitter into a deck-building rougelike.

[Original demo](https://twitter.com/rebane2001/status/1988251704437911664)

## Installing

You can either use the extension or userscript. Extension is work in progress so grab the userscript from [here](https://github.com/rebane2001/TweetsAgainstHumanity/raw/refs/heads/mane/extension/tweets-against-humanity.user.js) and install it with a userscript manager (such as Violentmonkey (recommended) or Tampermonkey).
